Instructions for running NChess Ranger.

Backend requirements:

Installations of django and dependencies.
Step 1: pip install django
Step 2: pip install djangorestframework
Step 3: pip install django-cors-headers
Step 4: cd n-chess-ranger/backend
Step 5: python manage.py runserver

Leave the server running in the background.


Frontend requirements:

Download Node.js from their website: https://nodejs.org/en/download  Verify installation by running these two commands:     node -v
    npm -v
Step 1: cd n-chess-ranger/frontend
Step 2: npm install (for downloading dependencies)
Step 3: npm run dev (to start the server)

Now you should be able to open the running local server on your browser.
